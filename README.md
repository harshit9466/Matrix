## Event Details

- **Event Name:** MATRIX 2026
- **Date:** February 13, 2026
- **Organizer:** School of Management, Vel Tech University
- **Registration Fee:** Rs. 150
- **Registration Deadline:** February 26, 2026

## About MATRIX

MATRIX is an Intercollegiate Management Festival that nurtures future business leaders by blending theoretical insights with practical problem-solving. Through activities like case competitions, marketing challenges, and management quizzes, students can showcase their skills and enhance their strategic thinking.

MATRIX highlights cultural events that honor creativity in music, dance, and the arts, creating a vibrant atmosphere. Ultimately, MATRIX serves as a hub where innovative ideas thrive, effective strategies are developed, and students evolve into the business leaders of tomorrow.

## Features

- **Responsive Design:** Mobile-friendly layout that adapts to all screen sizes
- **Interactive Event Cards:** Click-to-view popup system for event details and rules
- **Smooth Animations:** WOW.js animations and smooth scrolling
- **Image Slider:** Swiper.js powered hero slider
- **Tab Navigation:** Easy navigation between event categories and rules

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/harshit9466/Matrix.git
   cd Matrix
   ```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Using Python 3
     python -m http.server 8000

     # Using PHP
     php -S localhost:8000

     # Using Node.js (http-server)
     npx http-server
     ```

3. **Access the website:**
   - Navigate to `http://localhost:8000` in your browser
